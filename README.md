# Website - Hugo

Personal Website built with [HUGO](https://gohugo.io/)

Live Version can be seen at [carlossousa.tech](https://carlossousa.tech)


## Stack

- [HUGO](https://gohugo.io/) for Static Website Generation

- [Woodpecker](https://woodpecker-ci.org/) for the CI/CD pipelines

- [Docker](https://www.docker.com/)(-compose) for Deployment
